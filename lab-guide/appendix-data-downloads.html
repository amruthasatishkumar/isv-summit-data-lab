How to Upload These into Your Lakehouse

  1. Download the zip above (or grab individual files) and extract to a folder on your local machine. You should end up with facility_catalog.parquet next to a facilities/ folder containing five JPGs.
  2. In the Fabric portal open lh_urbanpulse_bronze (or whatever name you gave your Bronze Lakehouse in M2).
  3. In the Explorer pane right-click Files → Upload → Upload files and pick facility_catalog.parquet.
  4. Right-click Files again → Upload → Upload folder and pick the facilities folder containing the five JPGs.
  5. Right-click facility_catalog.parquet → Load to tables → New table and accept the auto-detected schema. Name the table facility_catalog (or whatever you prefer).
✓
After this you'll have one Delta table (facility_catalog) and one folder of binary assets (Files/facilities/) - exactly the same end state as the OneLake-shortcut path in M2 Option A.
